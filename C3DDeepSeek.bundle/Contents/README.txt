C3D DeepSeek AI - Assistente IA para Civil 3D 2026
========================================================

Este plugin integra a API DeepSeek ao AutoCAD/Civil 3D 2026.

COMANDOS DISPONIVEIS:
  DEEPSEEK  - Abre o painel de chat lateral com DeepSeek
  DSASK     - Envia uma pergunta direta via linha de comando
  CONFIGDS  - Configura/atualiza a chave API DeepSeek

CONFIGURACAO:
  A chave API fica no arquivo .env na pasta do plugin:
  DEEPSEEK_API_KEY=sk-sua-chave-aqui

EXEMPLOS DE USO:
  - "Como criar uma superficie TIN a partir de pontos?"
  - "Qual comando para gerar curvas de nivel?"
  - "Explique o fluxo para criar um corredor"
  - "Qual o atalho para editar o estilo de uma label?"

O DeepSeek pode sugerir comandos que serao executados
diretamente na linha de comando do Civil 3D.
